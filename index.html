What is Typo Resolver

  • 覺得網站上的錯字讓你渾身不舒服嗎？
  • 總是想要改掉錯字卻無從下手嗎？
  • 想要幫網站主人修改錯字嗎？

讓 Google 小工具 Typo Resolver 來幫你

How to use Typo Resolver

  1. 將擴充小工具安裝在您的Chrome瀏覽器中 前往Google小工具下載
  2. 將錯字反白標示出來
  3. 按下右鍵點選> 改錯> ，並再彈出視窗輸入想修正的文字
  4. 最後送出回報，請點選feed back
